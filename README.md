# Synapse Studio | Portfolio Architecture

A premium, cinematic portfolio experience for Sharukh H, built with a focus on liquid motion, editorial typography, and AI-powered content generation.

## 🚀 Tech Stack

### Frontend & Core
- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router, Server Components)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/) (Radix Primitives)
- **Icons**: [Lucide React](https://lucide.dev/)

### Motion & 3D Environment
- **3D Engine**: [Three.js](https://threejs.org/) (Custom vertex displacement & physical materials)
- **Animation Orchestration**: [GSAP](https://gsap.com/) (ScrollTrigger, Timeline, Parallax)
- **Component Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scrolling**: [Lenis.js](https://lenis.darkroom.engineering/) (Physics-based momentum scrolling)

### Generative AI
- **AI Toolkit**: [Firebase Genkit](https://firebase.google.com/docs/genkit)
- **LLM**: Google Gemini 1.5 Flash
- **Features**: 
  - `ai-project-description-generator-flow`: A custom Genkit flow designed to transform technical project details into professional, compelling portfolio copy.

### Infrastructure & Performance
- **Hosting**: [Firebase App Hosting](https://firebase.google.com/docs/app-hosting)
- **Optimization**: Next/Image (LCP optimized with priority & adaptive sizing)
- **Hydration Safety**: Client-side deferred execution for browser-specific APIs (Three.js/GSAP).

## 🛠️ Key Features
- **3D Liquid Soul**: A living, reactive background that breathes and morphs based on vertex noise.
- **Locked Works Exhibition**: A horizontal project showcase with progress tracking and manual navigation.
- **Editorial Typography**: Character-level pop reveals and line-by-line paragraph staggering.
- **The Odyssey**: A scroll-synced career timeline that draws its own path dynamically.
