# **App Name**: Synapse Studio

## Core Features:

- Cinematic Hero & Navigation: An immersive, full-screen introductory section featuring Sharukh H's name, title, tagline, an animated scroll indicator, and smooth-scrolling navigation across the site.
- Dynamic Portfolio Showcases: Interactive display sections for featured projects (with images, titles, descriptions, and tech stacks), a creative design gallery with a click-to-enlarge preview, and certification cards.
- Skills & Experience Timeline: Visually engaging sections highlighting technical skills with animated progress bars within glassmorphism cards and a glowing vertical timeline illustrating the professional journey.
- Interactive Contact & CTA: A glassmorphism contact form with fields for name, email, and message, featuring glow-on-focus inputs, alongside a prominent call-to-action section with magnetic buttons.
- AI Project Description Tool: An internal tool that helps Sharukh H to automatically generate concise and compelling project descriptions based on key project details and technologies used, streamlining content creation for the portfolio.
- Responsive Glassmorphism UI: A fully responsive user interface ensuring optimal viewing across all device sizes, consistently applying a dark-mode glassmorphism aesthetic with blurred backgrounds, subtle transparency, and soft borders to all interactive elements and content cards.
- Premium Motion & Micro-interactions: Implement a suite of advanced motion graphics including smooth scrolling, scroll-triggered animations (GSAP + ScrollTrigger), parallax effects, a custom cursor, magnetic button interactions, and elegant hover effects.

## Style Guidelines:

- Background color: Deep Space (#0B0B0F), providing a very dark, futuristic base with subtle blue-purple undertones that enhance other colors without being distracting.
- Primary highlight: Electric Blue (#3B82F6), a vibrant and energetic hue used for primary interactive elements, important textual highlights, and calls-to-action to grab attention.
- Secondary accent: Cyber Grape (#8B5CF6), a rich purple hue chosen to complement the primary blue, frequently utilized in animated gradients, secondary accent elements, and subtle background particles to add depth and sophistication.
- Headline and body font: 'Space Grotesk' (sans-serif), selected for its modern, slightly techy, and cinematic aesthetic, allowing for large, impactful headlines while maintaining excellent readability for descriptive text.
- Use clean, minimalist, and vector-based icons that seamlessly integrate with the dark, futuristic, and polished theme. Icons on interactive elements should feature soft glow highlights on hover.
- A minimalist, modular layout with distinct full-screen sections and ample modern spacing. Glassmorphism cards with a blurred background effect, soft borders, and subtle transparency are fundamental to the content containers.
- Implement a 'premium motion design' experience including smooth scrolling with Lenis.js, GSAP with ScrollTrigger for diverse animations (cinematic entrances, scroll reveals, text reveals, image tilts, card expands/flips), subtle parallax motions, and dynamic micro-interactions such as magnetic buttons and hover glows.
